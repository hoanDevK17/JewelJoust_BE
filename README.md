# Jewelry Auction Management System - Back-end

## Introduction

This project is the back-end part of an online jewelry auction management system. It provides the API and business logic for managing auctions, users, jewelry items, and transactions.

## Features

- User authentication and authorization (token-based and non-token-based)
- Managing jewelry auction requests from sellers
- Managing auction process for buyers
- Auction session management
- Jewelry management
- Transaction fee management
- Dashboard for statistics

## Tech Stack

- Spring Boot
- Spring Data JPA
- Spring Security
- Hibernate
- MySQL (or any preferred RDBMS)
- Maven

## Getting Started

### Prerequisites

- Java 11 or higher
- Maven 3.6.x or higher
- MySQL (or any preferred RDBMS)
